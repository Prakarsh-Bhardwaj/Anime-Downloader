# Anime Downloader w/ IDM
This folder contains 2 files -  
  
    Animed.py - anime downloader without multithreading.  
    Animedfaster.py - anime downloader with multithreading.  
  
Both download anime from gogoanime.com and use IDM.  
  
## Usage
Without multitreading-  
```bash
$python animed.py <name of anime> <starting ep to download> <ending ep to download>
```  
With multitreading-  
```bash
$python animedfaster.py <name of anime> <starting ep to download> <ending ep to download>
```
